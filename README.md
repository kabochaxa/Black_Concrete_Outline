# Black_Concrete_Outline
黒コンクリートを見やすくして、設置ミスを減らします。

# ダウンロード方法

右のReleasesからLatestと緑で書かれたものをクリックします。
その後Black_Concrete_Outline.zipをクリックすることでファイルをダウンロードできます。
ダウンロードしたファイルを解凍せずにリソースパックフォルダへ入れてください。
